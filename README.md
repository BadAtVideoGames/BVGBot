# BVGBot
The Official Bad At Video Games Bot

## Setting up the application
### Clone the repo
```bash
git clone https://github.com/BadAtVideoGames/BVGBot.git
cd BVGBot
```

### Create config file
```bash
cp bvgbot.yml.example bvg_bot.yml
```

Edit the email and password in the config file you just created

### Setup the environment
```bash
bundle
```

## Running the application
```bash
ruby run.rb
```
